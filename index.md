# Добро пожаловать в проект «Цветочная оранжерея»! 🌸

Этот сайт создан командой **Supermodels** в рамках Лабораторной работы №4.

## 👥 Наша команда
- **Григорьева Александра** — Тимлид, системный аналитик
- **Кудинова Елизавета** — Разработчик БД, программист

## 📋 О проекте
Наша цель — создать удобное консольное приложение на языке C для управления автоматизированной оранжереей. Программа позволяет:
* Вести учет видов растений.
* Контролировать температуру и влажность.
* Отслеживать графики полива.
* Хранить фотографии цветов в базе данных SQLite.
